# Mini-Dil-Öğretici
Kelime Translate Özelliği
Kişisel Skor Kaydetme Özelliği
Dinleme Ve Kayıt Alma Özelliği
Yanlış ile Doğruyu Ayırt Etme Özelliği

Bu mini oyunda sizden öncelikle bir seviye seçmeniz isteniyor. Seçtiğiniz seviyeye atanmış kelimelerden biri rastgele olarak ekranınıza geliyor ve ingilizcesini söylemeniz isteniyor.Verdiğiniz cevapdoğru ise 5 puan kazanıyorsunuz. Eğer cevabınız yanlışsa 5 canınızdan birini kaybediyorsunuz. Ayrıca puanınız 20 veya 20den fazla olursa size tekrardan sevye seçme hakkı veriliyor.
